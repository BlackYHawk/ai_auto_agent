# Quickstart: GUI增强功能

## 功能概览

1. **项目列表** - 首页显示所有项目卡片
2. **项目详情** - 查看完整项目信息
3. **章节查看** - 阅读章节内容
4. **小说导入** - 导入TXT文件创建项目
5. **模型配置** - 设置LLM模型

## 测试场景

### 场景1: 查看项目列表

```bash
# 运行GUI
cargo run

# 预期：首页显示 projects/ 目录下所有项目
```

### 场景2: 导入小说

```bash
# 准备TXT文件（UTF-8编码）
# 章节格式：第1章 标题
#          正文内容...

# 在GUI中点击"导入项目"
# 选择TXT文件
# 预览分割结果
# 确认创建
```

### 场景3: 切换模型

```bash
# 方式1: 全局配置
# GUI > 设置 > 选择默认模型 > 保存

# 方式2: 项目级配置
# 项目详情 > 点击模型 > 选择 > 保存
```
